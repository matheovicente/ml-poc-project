# Assignment 2 - Exploration des données

Le dataset brut contient les observations journalières PortWatch pour 28
chokepoints maritimes.

Premières questions d'exploration :

- quels chokepoints ont le trafic moyen le plus élevé ?
- lesquels concentrent le plus de tankers ?
- lesquels ont la plus forte capacité totale ?
- lesquels présentent les perturbations les plus fréquentes ?

Les visualisations principales sont sauvegardées dans `plots/`.
