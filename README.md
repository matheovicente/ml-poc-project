# Maritime Straits ML Project

## Current Deliverable

This repository currently contains the first project deliverable:

- `deliverables/assignement1.md`: business case, dataset description, and initial ML approach.
- `notebooks/01_portwatch_exploration.ipynb`: first exploratory analysis of the PortWatch maritime chokepoints dataset.
- `data/raw/`: raw PortWatch data used by the notebook.

## Project Topic

The project studies the strategic importance of maritime chokepoints and the potential impact of temporary closures on global maritime traffic.

The first exploration focuses on:

- identifying the main chokepoints in the dataset;
- comparing total ship traffic;
- comparing tanker exposure;
- focusing on strategic passages such as the Strait of Hormuz, Malacca Strait, and Taiwan Strait.

## How to Open the Notebook

From the repository root:

```bash
jupyter notebook notebooks/01_portwatch_exploration.ipynb
```

or:

```bash
jupyter lab notebooks/01_portwatch_exploration.ipynb
```
